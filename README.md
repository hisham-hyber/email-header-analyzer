# Email Header Analyzer (OSINT Dashboard)

A simple browser-based tool to analyze email headers and extract metadata such as:
- Sender / Receiver
- Subject decoding (MIME)
- Email route (Received headers)
- Server providers (Google, SendGrid, etc.)

## Demo
https://hisham-hyber.github.io/email-header-analyzer/
## Usage
1. Open the tool
2. Paste email headers
3. Click Analyze

## Purpose
This project demonstrates how email metadata can expose infrastructure and reduce anonymity, even when encryption is used.
